# **DIGITAL IMAGE PROCESSING : ЛАБОРАТОРНІ**
---
## **ЧАСТИНА 2. 3D редактор BLENDER. Створення ігрового додатку**

## **Мета.**
### Оволодіти практичними навичками використання графічного редактора Blender для побудови 3D об'єктів  ігрових додатків. Сформувати практичні навички створення простого ігрового графічного середовища сучасними засобами для  Win, Android, HTML.
## **Наскрізне завдання.**
### За допомогою обраних інструментальних засобів створити графічне середовіще ігрового додатку.
